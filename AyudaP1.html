Proceso N° 1

Descargar datos de Bases de Datos locales

Este proceso tiene como fin Generar un archivo CSV con datos de estudiantes inscritos en el lapso que no tengan correo institucional.

El archivo CSV a generar tendrá la siguiente estructura: nombres,apellidos,cedula,sede,carrera,correoalternativo,estatuscorreo

El formato del nombre del archivo CSV es: Siglas de Sede + Fecha en formato aammddhhmm.csv
Ejemplo: VRB1005101528.csv

Las siglas de los vicerrectorados y nucleos son:

Siglas Sede
VRB Vicerrectorado Barquisimeto
NCa Núcleo Carora
VRP Vicerrectorado puerto Ordaz
VRL Vicerrectorado Caballero Mejias
NCh Núcleo Charallave
NGu Núcleo Guarenas

Para generar este archivo se deben seguir los siguientes pasos
1.-Seleccionar Sede y Lapso
2.-Pulsar el botón Descargar.
3.-Los Archivos generados por este proceso servirán de entrada al Proceso N° 2.
