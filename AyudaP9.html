Proceso N° 9

Actualizar datos de Bases de Datos locales

Este proceso tiene como fin actualizar los datos de correo institucional del estudiante (CORREO_INS, CLAVE_INS_T, CLAVE_INIC_CORREO ).

Los archivo CSVs a utilizar como fuente tendrá la siguiente estructura: idcorreo,password,sede,cedula

El formato de los nombres de los archivos CSV son: G que lo identifica como un archivo de grupo + separador '-' + siglas de Sede + separador '-' + siglas de Carrera + separador '-' + Fecha en formato aammddhhmm + Correlativo que indica grupos de 250 estudiantes por archivos de una misma carrera.csv
Ejemplo: G-VRB-IEL-110520101.csv

Las siglas de los vicerrectorados y nucleos son:

Siglas Sede
VRB Vicerrectorado Barquisimeto
NCa Núcleo Carora
VRP Vicerrectorado puerto Ordaz
VRL Vicerrectorado Caballero Mejias
NCh Núcleo Charallave
NGu Núcleo Guarenas

Las siglas de las carreras o especialidades son:


Siglas Carrera

Para generar actualizar los datos se deben seguir los siguientes pasos
1.-Descomprimir previamente el/los archivos generados en los Procesos N° 8
2.-Pulsar el botón Examinar, para Seleccionar un CSV especifico.
3.-Pulsar el botón Cargar, para inicializar el proceso.
